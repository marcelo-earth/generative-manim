# Animo

[Animo](https://animo.video) is a lightweight desktop application that converts natural language descriptions into animated videos using AI and the Manim engine.

## What it does

Animo enables educators and students to create professional mathematical and educational animations without coding expertise. Users describe what they want to animate in plain English, AI generates the Manim code, and the application renders it locally into video.

## How it works

1. Describe your animation concept in everyday language
2. AI generates Manim code based on that description
3. The application renders the animation locally and exports MP4 files

## Key features

- **AI-Powered Creation** — Describe what you want to animate and let AI generate the code
- **Local Rendering** — All processing happens on your machine, no cloud uploads required
- **Real-time Preview** — Live video rendering as animations are created
- **Template Library** — Pre-built templates covering physics, algebra, astronomy, and chemistry
- **Export Options** — Multiple formats (MP4, GIF, PNG sequences) and aspect ratios
- **Code Ownership** — Full access to generated Manim code for further customization
- **Lightweight** — Runs in under 7 MB

## Technology

- Built on the open-source [Generative Manim](https://github.com/marcelo-earth/generative-manim) project
- Integrates with Claude Code and OpenAI Codex for code generation
- Native desktop client for macOS (Apple Silicon and Intel) and Windows
- Requires Python 3.9+ with Manim and FFmpeg installed
